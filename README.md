# SwpTextView


##### 封装 带 Placeholder 的 TextView
---
#####导入：
```
手动导入：

SwpTextView 文件夹 导入 项目 中
#import "SwpTextView.h"
```
---

```ruby

CocoaPods 导入:

pod search SwpTextView

pod 'SwpTextView'

#import <SwpTextView/SwpTextView.h>

```
